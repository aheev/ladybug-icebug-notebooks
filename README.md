# Ladybug + Icebug Star Wars Notebook

This workspace contains a reproducible `uv` project and a notebook demo for the
Ladybug nightly Arrow query path feeding Icebug graph analytics.

Run it with:

```bash
uv run jupyter notebook ladybug_icebug_starwars.ipynb
```

Or execute it from the CLI:

```bash
uv run jupyter nbconvert --to notebook --execute ladybug_icebug_starwars.ipynb --inplace
```
